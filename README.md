# site-zh-notes

本仓库用于归档和发布一批独立 HTML 页面。这些页面内容彼此分离，不隶属于任何特定网站或域名，作为一份静态资料合集存在。

## 项目简介

site-zh-notes 是一个以 HTML 文档为载体的信息存档仓库。每个页面均为自包含的独立文件，可单独打开浏览，无需依赖后端服务或数据库。仓库的主要目的是长期保存与展示这些页面内容，方便查阅与分发。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库主页 / 导航页
├── pages/             # 存放所有独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 页面依赖的静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- **index.html**：可选的主页，用于展示归档内容列表或简要说明。
- **pages/**：核心目录，每个 HTML 文件即一个独立页面。
- **assets/**：存放页面共用的或引用的静态资源文件。

## 页面归档说明

- 所有页面均为静态 HTML，无服务器端依赖。
- 页面间彼此独立，互不依赖。
- 文件名建议使用英文小写、连字符分隔，保持清晰可读。
- 如需引用本地资源，请将资源放置于 `assets/` 对应子目录下，并使用相对路径加载。

## 维护说明

- 本仓库由维护者不定期更新，添加新页面或修正已有内容。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正。
- 所有页面内容应保持中立、客观，不包含营销、推广或诱导性信息。
- 仓库不保证页面内容长期可用或指向外部链接的有效性。

## 使用方式

1. 克隆或下载本仓库。
2. 直接双击 `pages/` 下的 HTML 文件，即可在浏览器中查看。
3. 若部署至静态服务器（如 GitHub Pages），可通过 `index.html` 或直接访问各页面 URL。
